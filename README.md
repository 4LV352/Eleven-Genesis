# Eleven Genesis

Manager de futebol mobile-first em HTML, CSS e JavaScript puros.

## Estrutura

```text
index.html
css/
  tokens.css
  components.css
  pages.css
js/
  game.js
  main.js
```

## Como Jogar

Abra `index.html` no navegador.

O jogo salva automaticamente no `localStorage`.

## Sistemas Atuais

- Splash screen com identidade Eleven Genesis.
- Dashboard do clube com próxima partida, forma recente e notícias.
- Simulação de partidas rodada a rodada.
- Classificação da liga com jogos de clubes rivais.
- Elenco com atributos, moral, salários e venda de jogadores.
- Mercado com prospecção e contratação.
- Finanças com saldo, receitas, folha salarial e extrato.
- Nova carreira com limpeza do save local.

## Direção Visual

- Verde gramado, dourado troféu, preto estádio e marfim refletor.
- Dark mode por padrão.
- Tipografia: Bebas Neue, Inter e Oswald.
- Layout mobile-first otimizado para 375px.

## Regras Técnicas

- Sem frameworks.
- Sem build.
- Sem dependências.
- Persistência local via `localStorage`.
